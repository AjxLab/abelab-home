Smart speaker working from my home.


## Demo
[demo video](https://streamable.com/e/njvovx)


## Requirements
* Raspbian 4.19
* Python 3.7


## Usage
### Use Docker
```sh
$ docker-compose up -d
```
### Not Use Docker
```sh
$ ./abelab-home.py
```


## Installation
```sh
$ git clone <this repo>
$ cd <this repo>
```
